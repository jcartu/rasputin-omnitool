# Code of Conduct

This project follows a simple rule: **be civil and substantive**.

## Expected behavior

- Critique code, not people.
- Assume good faith; ask before asserting.
- Stay on topic. This is a small repo with a narrow scope.
- If you disagree with a design decision, say so concretely and propose an alternative.

## Unacceptable behavior

- Personal attacks, harassment, or discrimination.
- Spam or off-topic promotion.
- Bad-faith debate (sealioning, JAQing off, etc.).

## Enforcement

The maintainer (Joshua Cartu, [@jcartu](https://github.com/jcartu)) reserves the right to lock threads, close issues, or block users without further explanation if this code of conduct is violated.

If you see a problem, open an issue or contact the maintainer directly.
